Installation Guide

This guide explains how to install Node.js on a Windows system.

Prerequisites

  • Windows 10 or later
  • Administrator access
  • Internet connection

Download Node.js

  1. Open your browser.
  2. Go to the official Node.js website.
  3. Download the LTS version.

Install Node.js

  1. Locate the downloaded installer file.
  2. Double-click the file to launch the installer.
  3. Click Next.
  4. Accept the license agreement.
  5. Click Next through the setup steps.
  6. Click Install.

Verify Installation

Open Command Prompt and run:

node -v

If installed correctly, the version number is displayed.

Troubleshooting

Node command not recognized

Restart your system and try again.

Installation fails

Ensure you have administrator privileges and retry installation.

Next Steps

You can now install packages using npm:

npm install express
